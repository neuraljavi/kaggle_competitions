# Competencias de Kaggle / Kaggle Competitions

📁 Repositorio personal para documentar mi participación en competencias de Kaggle.

## 📌 Descripción

Este repositorio contiene mis soluciones y enfoques a distintas competencias de Kaggle. Comenzando con una competencia de introducción (*Getting Started*), iré subiendo progresivamente notebooks, scripts y modelos utilizados en cada una de las competencias en las que participe.

**Importante**: Este repositorio **no incluye datasets**. Todos los datos deben obtenerse directamente desde la plataforma de Kaggle, respetando sus términos de uso.

## 📂 Estructura

Cada competencia tendrá su propia carpeta con:
- 📒 Notebooks (Jupyter o Kaggle)
- 🧠 Modelos entrenados (si aplica)
- 📝 Notas o scripts auxiliares
